⚙️ How the Automation Works

  • Reads next unprocessed post from the Tumblr export (chronological order)
  • Extracts Chinese title + body paragraphs
  • Translates to English using Claude Sonnet — preserves pastoral tone, handles idioms naturally
  • Generates a 145-character English social preview hook
  • Creates a 1200×630 cover image with "Pastor Pei 夏牧師" branding in navy + gold
  • Uploads cover to Substack CDN
  • Posts bilingual draft (Chinese body → "English Translation" heading → English body → Subscribe widget)
  • Logs everything to progress.json so no post is ever processed twice
# Run from the Pastor Pei project folder
python3 automate.py                    # 2 posts, Sonnet (default)
python3 automate.py --n 1           # 1 post at a time
python3 automate.py --model haiku    # cheaper, slightly less polished
